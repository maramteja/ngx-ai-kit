Open Source Angular Library

ngx-ai-kit

Drop-in AI components for Angular — chat widget, autocomplete & streaming text.
Works with any LLM backend.

npm install ngx-ai-kit GitHub
💬

Chat Widget

Full-featured streaming chat UI with auto-scroll, message history, and welcome messages.

⚡

AI Autocomplete

Debounced AI suggestions with keyboard navigation. Reactive Forms compatible (CVA).

✨

Streaming Text

Inline streaming text with a blinking cursor. Perfect for generative content.
